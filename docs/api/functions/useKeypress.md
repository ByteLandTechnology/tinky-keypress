# Function: useKeypress()

> **useKeypress**(`onKeypress`, `options`): `void`

A hook that listens for keypress events from stdin.

## Parameters

### onKeypress

[`KeypressHandler`](../type-aliases/KeypressHandler.md)

The callback function to execute on each keypress.

### options

Options to control the hook's behavior.

#### isActive

`boolean`

Whether the hook should be actively listening for input.

## Returns

`void`
