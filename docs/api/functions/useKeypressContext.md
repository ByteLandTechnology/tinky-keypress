# Function: useKeypressContext()

> **useKeypressContext**(): [`KeypressContextValue`](../interfaces/KeypressContextValue.md)

Hook to access the Keypress context.
Allows components to subscribe and unsubscribe to keypress events.

## Returns

[`KeypressContextValue`](../interfaces/KeypressContextValue.md)

The Keypress context value containing subscribe and unsubscribe methods.

## Throws

Error if used outside of a KeypressProvider.
